# Manajemen Strategik — MSK304

Course support system for **MSK304 Manajemen Strategik** (Magister Akuntansi, STIE YKPN TA 2025–2026).
Authored by Dzaki Muhammad Yusfian (01079).

14 tatap-muka, four HTML outputs per week (Study Guide, Main Summary, Article Analysis, Indonesian Company Examples), plus cross-week Visual Companion and Master Print Bundle. Primary text: Thompson/Peteraf/Gamble/Strickland *Crafting and Executing Strategy* 7e (TPGS); supplements: Henry *Understanding Strategic Management* 4e and assigned journal articles.
