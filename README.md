# MST304 — Manajemen Strategik Kontemporer

**Program Magister Akuntansi · STIE YKPN · Semester 1**  
Dzaki Muhammad Yusfian (01079)

🌐 **Hub Page:** [dmy-cmyk.github.io/msk304-course-materials](https://dmy-cmyk.github.io/msk304-course-materials/)

Course support system for **MSK304 Manajemen Strategik** — 14 tatap-muka, four HTML outputs per week (Study Guide, Main Summary, Article Analysis, Indonesian Company Examples), plus cross-week Visual Companion and Master Print Bundle. Primary text: Thompson/Peteraf/Gamble/Strickland *Crafting and Executing Strategy* 7e (TPGS); supplements: Henry *Understanding Strategic Management* 4e and assigned journal articles.
